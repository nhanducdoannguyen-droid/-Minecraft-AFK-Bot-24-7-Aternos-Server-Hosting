Hướng Dẫn Nhanh

Nhấn để mở/đóng — 4 bước setup cho người mới
▼
1
Bật Server & Setup Cracked
Đăng nhập aternos.org → bật server. ⚠️ BẮT BUỘC: Vào mục Options (Tùy chọn) → Bật Cracked (Bản bẻ khóa) lên. Nếu không bật, bot sẽ không vào được!
2
Điền Thông Tin Server
Điền Tên Bot, Server Host (phần trước dấu :), Server Port (phần sau dấu :), và Phiên Bản MC ở mục bên dưới.
3
Bật Tính Năng (Tùy Chọn)
Discord Webhook — nhận thông báo về Discord. LoginSecurity — bật nếu server yêu cầu /login. Chống AFK — nên bật để tránh bị kick.
4
Nhấn "Lưu & Khởi Động Lại Bot"
Bot sẽ tự kết nối đến server. Khi thành công, trạng thái chuyển sang 🟢 Online. Xong!
🎮

Thông Tin Server

Cấu hình kết nối Minecraft
⚠️

Lưu ý cho server Aternos: Bạn bắt buộc phải vào mục Options (Tùy chọn) trên trang web Aternos và bật Cracked (Bản bẻ khóa) lên. Nếu không bật, bot sẽ bị lỗi "Disallowed connection" hoặc "Not authenticated with Minecraft.net".

💬

Discord Webhook

Gửi thông báo về Discord
Tạo webhook trong Discord Server Settings → Integrations → Webhooks
Dùng proxy nếu deploy trên HuggingFace (bị chặn gửi trực tiếp đến Discord)
ℹ️

Khi bật, bot sẽ gửi thông báo về Discord khi: kết nối/ngắt kết nối, player join/leave, phát hiện hack/cheat, và trạng thái bot.

🏃

Chống AFK

Giả lập player thật, tránh bị kick
🎭

Bot sẽ tự động di chuyển, nhìn xung quanh, nhảy, vung tay, đổi hotbar, mở inventory... giống player thật để tránh bị hệ thống anti-AFK phát hiện và kick.

🔐

LoginSecurity Plugin

Tự động đăng nhập/đăng ký
Mật khẩu để bot tự đăng ký & đăng nhập vào server có plugin LoginSecurity
🔑

Cách hoạt động: Lần đầu vào server → bot tự gửi /register <pass> <pass> để tạo tài khoản. Các lần sau → bot tự gửi /login <pass> để đăng nhập. Bot tự phát hiện yêu cầu từ plugin.

☁️

HuggingFace Spaces

Cấu hình qua Environment Variables
💡

Bạn cũng có thể cấu hình qua HuggingFace Secrets (Settings → Variables and secrets). Env vars sẽ override config từ web panel. Các biến hỗ trợ:

MC_HOST · MC_PORT · MC_USERNAME · MC_VERSION
DISCORD_WEBHOOK_URL · WEBHOOK_PROXY · WEBHOOK_ENABLED
BOT_PASSWORD · ANTI_AFK
